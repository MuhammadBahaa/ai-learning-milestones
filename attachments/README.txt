Attachment images referenced from milestones.js live here.

Drop an image in this folder (e.g. welcome-to-claude-code.png) and point a
milestone's attachment at it:

  attachment: {
    file: 'welcome-to-claude-code.eml',
    note: 'a playful caption',
    image: 'attachments/welcome-to-claude-code.png',
  },

The card then shows a clickable thumbnail that opens the image full-size.
If the referenced image is missing, the card just shows the plain chip.
